# 🎬 Movie Ticket Booking Website

A modern **Movie Ticket Booking Website** built using **HTML, CSS, and JavaScript**.  
This project allows users to browse movies and book tickets through an interactive booking form.

---

## 🚀 Features

- 🎥 Movie showcase section
- 🪑 Ticket booking popup form
- 📱 Fully responsive design
- ✨ Smooth animations and hover effects
- 📜 Movie listing carousel
- 📩 Contact section
- 🔗 Social media footer
- 🎨 Modern UI with gradient and glow effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons

---



---

## 📸 Preview

### Home Section
Displays the featured movie and booking option.

### Booking Section
Shows available movies with booking buttons.

### Contact Section

## 🎟️ Booking System

When a user clicks **Book Ticket**, a popup form appears where they can enter:

- Name
- Email
- Ticket Count

The website is fully responsive and works on:
